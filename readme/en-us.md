## Introduction
ApiCat is an API framework that helps webmasters quickly deploy various APIs.
## Use
### Installation  
run
```bash
pip install apicat
```
Download the apicat package.
### Initialization. 
Go to the project directory and run
```bash
apicat init
```
After that, three files will appear in your directory,
### Run. 
In the project directory, run
```bash
apicat start
```
At this point, you have successfully launched a blank API platform.
